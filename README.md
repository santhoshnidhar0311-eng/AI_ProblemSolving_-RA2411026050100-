# 🎮 Tic Tac Toe Game

## 📌 Problem Description
Develop an interactive Tic Tac Toe game that allows two players to play in a browser. The game should detect a winner or a draw based on standard rules.

---

## ⚙️ Algorithms Used

1. Board Representation:
   - Use an array of 9 elements to represent the grid

2. Turn Management:
   - Alternate turns between Player X and Player O

3. Win Checking:
   - Check all winning combinations:
     Rows, Columns, Diagonals

4. Draw Condition:
   - If all cells are filled without a winner → Draw

---

## ▶️ Execution Steps

1. Clone the repository:
   git clone https://github.com/your-username/tic-tac-toe-interactive-game.git

2. Open the project folder

3. Run:
   Open index.html in browser

4. Play:
   Click cells to mark X or O
